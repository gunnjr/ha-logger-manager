# Step 4 — Selection Basket (Chips) + Remove

## Objective
Show selected items as chips in a basket; allow removing and clearing.

## Prompt to Copilot
- Add a selection area (right side or bottom on mobile).
- Render chips (label + ✕) for each selected item.
- Show **Selected: {count}** badge.
- ✕ removes from selection and returns it to results.
- Include “Clear all” for selection.

## Acceptance Checklist
- Chips reflect current selection accurately.
- Removing a chip returns it to the results list.
- Count updates correctly.
